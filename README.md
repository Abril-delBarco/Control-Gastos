Aplicación de escritorio desarrollada en Python para gestionar finanzas personales: registrar gastos, marcarlos como pagados o pendientes, filtrarlos por mes y visualizar un balance general.
Este proyecto lo desarrollé aplicando Programación Orientada a Objetos (POO), como forma de practicar y afianzar estos conceptos.

Funcionalidades
- Agregar gastos con monto, descripción, categoría y fecha.
- Categorías predefinidas: Comida, Servicios, Diversión, Transporte, Ropa, Salud, Educación, Otros.
- Marcar como pagado un gasto pendiente.
- Editar gastos ya cargados.
- Eliminar gastos con confirmación.
- Filtrar por mes para ver solo los gastos de un período específico.
- Balance automático: total registrado, monto ya pagado y monto pendiente, con colores para diferenciarlos.
- Persistencia de datos: los gastos se guardan automáticamente en un archivo Gastos.json, por lo que no se pierden al cerrar la aplicación.

Python 3
Tkinter (interfaz gráfica)
JSON (almacenamiento local de los datos)
